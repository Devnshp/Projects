## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/ml-projects.git
   ```

2. Navigate to the folder:

   ```bash
   cd ml-projects
   ```

3. Open Jupyter Notebook:

   ```bash
   jupyter notebook
   ```

4. Run any `.ipynb` file
